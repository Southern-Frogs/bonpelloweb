Notas de versión

Lo que necesitas saber

Etapas iniciales del desarrollo: Recuerda que la aplicación aún se encuentra en las primeras etapas de desarrollo. Es posible que encuentres algunos errores, fallos o características faltantes.

 

Funciones limitadas: La versión alfa puede tener características limitadas en comparación con la versión final. Una de las principales características que nos distingue de otras aplicaciones o sitios web que analizan productos aprobados para el método curly girl es la capacidad de detectar errores en los ingredientes INCI. Esta función depende de que la aplicación conozca todos los posibles ingredientes de productos para el cabello y pueda identificarlos incluso con diferentes formas de escribirlos. Estamos trabajando arduamente para descubrir todas las variaciones, así que te pedimos paciencia mientras seguimos mejorándola.

 

Problemas de rendimiento: A medida que afinamos la aplicación, es posible que experimentes algunos pequeños problemas de rendimiento, como una carga lenta o una respuesta más lenta. No te preocupes, tus comentarios sobre el rendimiento nos ayudarán a solucionar estos problemas.

Reglas y Pautas del Programa Alpha

 

Actualizaciones de la aplicación: Cada vez que haya una nueva versión de la aplicación, la encontrarás en la lista de aplicaciones para actualizar en el Play Store. Puedes actualizarla manualmente, o si la configuración de tu teléfono lo permite, se actualizará automáticamente.

 

Reporte de errores: Si encuentras algún error, por favor envía una descripción a hello@bonpello.com. Sería genial si pudieras incluir una serie de pasos para reproducir el error y capturas de pantalla, si corresponde. Aquí te brindo un par de ejemplos de errores para darte una idea:

  • Un ingrediente rompe el reconocimiento de ingredientes. Por ejemplo, un ingrediente con un símbolo, como "C.I.", solía causar problemas.
  • La aplicación no se ve bien en tu dispositivo y las palabras se salen de sus contenedores.

 

Sé Constructiva: Al brindar tus comentarios, por favor, sé amable y constructiva. ¡Recuerda que también tengo sentimientos! ☺

 

Recompensas para los mejores colaboradores: Como muestra de nuestro agradecimiento, nos encantaría premiar a los colaboradores más destacados con regalitos relacionados con la app.

 

Sin presiones: Si no tienes mucho tiempo para invertir, ¡no te preocupes! Aún puedes usar la aplicación como lo harías normalmente. Tu uso y experiencia también son muy valiosos para nosotros.

 

Problemas conocidos: Puedes consultar una lista de los problemas ya identificado. Revisa esta página para ver si tu error ya ha sido reportado o si hay alguna solución alternativa disponible.
